GAME DESIGNER DOCUMENT

SINOPSIS
Tahun 2045. Organisasi militer gelap bernama SHADOW CORPS menguasai kompleks militer strategis dan menyandera data rahasia negara. Satu prajurit terbaik dikirim untuk menembus pertahanan mereka sendirian guna mengambil kembali data rahasia tersebut Agent K harus berjuang melawan para musuh dan kembali dengan misi berhasil serta selamat.
